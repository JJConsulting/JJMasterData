# JJLinkButton

Represents a clickable button.

## Usage


```csharp
@{
     var link = new JJLinkButton();
     link.Text = "JJ Consulting";
     link.IconClass = "fa-solid fa-home";
     link.UrlAction = "http://www.jjconsulting.com.br";
}

/// ...
@Html.Raw(search.GetHtml())
/// ...
```
